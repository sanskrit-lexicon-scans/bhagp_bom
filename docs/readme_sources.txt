Download the two volumes of Bombay edition at urls:
vol 1: https://www.digitale-sammlungen.de/en/view/bsb10211024
vol 2: https://www.digitale-sammlungen.de/en/view/bsb10625867

See below for 'details for volume 1' and 'details for volume 2'

These two downloaded into orig_unrotated directory as
 bombay_v1.pdf and bombay_v2.pdf.
 Size of each volume is approximately 600MB.

The pages of these two need to be rotated by 90-degrees for viewing.
rotation done using AdobeAcrobat 9.0 Document/RotatePages function
 and the results saved in tht 'orig' directory 
 as bombay_v1.pdf and bombay.v2.pdf
